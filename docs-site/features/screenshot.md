# 截图功能

PolarisDesk 提供了强大的截图功能，让你可以快速捕获屏幕内容并发送给 AI 分析。

## 截图方式

### 1. 快速截图 📷

快速截取整个屏幕：

- **快捷键**: `Ctrl/Cmd + Shift + A`（可在设置中自定义）
- **按钮**: 点击输入框左侧的 📸 图标
- **右键菜单**: 右键点击截图按钮切换模式

**使用场景**:

- 快速分享整个屏幕
- 捕获完整的应用界面
- 记录当前工作状态

### 2. 区域截图 ✂️

选择屏幕的特定区域进行截图：

- **快捷键**: `Ctrl/Cmd + Shift + X`（可在设置中自定义）
- **按钮**: 点击输入框左侧的截图图标（区域模式）
- **操作**: 拖动鼠标选择要截取的区域

**使用场景**:

- 只截取需要的部分
- 避免泄露敏感信息
- 聚焦特定内容

### 3. 自动截图

开启后，发送消息时自动截取整个屏幕：

1. 打开 **设置** > **应用设置**
2. 开启 **自动截图**
3. 发送消息时，如果没有附带任何媒体，会自动截图

**使用场景**:

- 需要 AI 看到你的屏幕
- 频繁需要截图的工作流
- 快速获取屏幕上下文

::: tip 提示
自动截图只在没有附带图片、视频或文档时触发。
:::

## 使用场景

### 代码调试

截取代码编辑器的屏幕，让 AI 帮你：

- 发现代码错误
- 优化代码结构
- 解释代码逻辑
- 提供改进建议

### 设计反馈

截取设计稿或网页，让 AI：

- 分析设计风格
- 提供改进建议
- 检查可访问性
- 对比设计规范

### 错误诊断

截取错误信息，让 AI：

- 解释错误原因
- 提供解决方案
- 给出调试步骤
- 推荐相关文档

### 学习辅助

截取学习材料，让 AI：

- 解释复杂概念
- 总结重点内容
- 回答相关问题
- 提供补充资料

## 快捷键

| 功能     | Windows/Linux  | macOS         |
| -------- | -------------- | ------------- |
| 快速截图 | `Ctrl+Shift+A` | `Cmd+Shift+A` |
| 区域截图 | `Ctrl+Shift+X` | `Cmd+Shift+X` |
| 粘贴图片 | `Ctrl+V`       | `Cmd+V`       |

::: tip 自定义快捷键
可以在 **设置** > **应用设置** > **快捷键** 中自定义截图快捷键。
:::

## 注意事项

### 支持的模型

需要使用支持视觉的多模态模型：

- **OpenAI**: GPT-4 Vision, GPT-4o
- **Anthropic**: Claude 3 (Opus, Sonnet, Haiku)
- **Google**: Gemini Pro Vision, Gemini 1.5
- **其他**: 支持视觉的多模态模型

::: warning 注意
如果使用不支持图片的模型（如 GPT-3.5），截图会被忽略。
:::

### macOS 权限

macOS 需要授予屏幕录制权限：

1. 打开 **系统设置** > **隐私与安全性** > **屏幕录制**
2. 找到 PolarisDesk
3. 勾选允许

### 图片限制

- **格式**: PNG
- **质量**: 无损压缩
- **大小限制**: 10MB
