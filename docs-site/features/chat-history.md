# 对话历史

自动保存所有对话记录。

## 查看历史

左侧边栏显示所有对话列表，点击切换对话。

## 搜索对话

在搜索框输入关键词搜索历史对话。

## 管理对话

### 新建对话

- 点击左上角 ➕ 按钮
- 快捷键 `Ctrl/Cmd + N`

### 删除对话

- 右键点击对话
- 选择"删除"

### 清空所有对话

设置 > 数据管理 > 清空对话历史

## 数据存储

所有对话存储在本地 SQLite 数据库中：

- Windows: `%APPDATA%/polaris-desk`
- macOS: `~/Library/Application Support/polaris-desk`
- Linux: `~/.config/polaris-desk`
