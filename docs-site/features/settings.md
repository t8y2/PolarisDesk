# 设置

配置应用的各项功能。

## AI 设置

### 选择服务商

- OpenAI
- Anthropic (Claude)
- Google (Gemini)
- DeepSeek
- Moonshot
- 其他

### 配置 API Key

输入对应服务商的 API Key

### 选择模型

根据服务商选择可用的模型

### 模型参数

- Temperature: 控制回复的随机性
- Max Tokens: 最大回复长度
