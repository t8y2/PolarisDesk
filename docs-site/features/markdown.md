# Markdown 渲染

AI 回复支持 Markdown 格式。

## 支持的语法

### 代码块

支持 100+ 编程语言的语法高亮

```python
def hello():
    print("Hello, World!")
```

### 数学公式

使用 KaTeX 渲染数学公式

行内公式：$E = mc^2$

块级公式：

$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

### 表格

| 列1   | 列2   |
| ----- | ----- |
| 数据1 | 数据2 |

### 其他

- 粗体、斜体
- 列表
- 引用
- 链接

## 代码操作

- 点击代码块右上角复制按钮复制代码
- HTML 代码可以预览效果
