# 更新日志

记录 PolarisDesk 的版本更新和新功能。

## 最新版本

### 新增功能

#### 🎭 人设预设系统

快速切换不同的 AI 角色和人设：

- **内置人设**: 编程专家、写作助手、学习导师等
- **自定义人设**: 创建你自己的 AI 角色
- **快速切换**: 在输入框直接切换人设
- **人设管理**: 在设置中管理所有人设

详见 [人设预设文档](../features/persona-presets.md)。

#### 💻 AI 命令行

AI 理解自然语言意图并生成执行系统命令：

- **自然语言**: 用人话描述想做什么
- **命令生成**: AI 自动生成对应的系统命令
- **安全执行**: 可选择手动确认或自动执行
- **跨平台**: 支持 Windows、macOS、Linux

详见 [AI 命令行文档](../features/ai-command.md)。

#### 🖥️ UI 树感知 (macOS)

自动捕获窗口的 UI 结构，让 AI 更懂你的界面：

- **自动捕获**: 发送消息时自动获取 UI 树
- **元素识别**: 识别按钮、文本框、标签等元素
- **上下文理解**: AI 能看到你的界面结构
- **辅助功能**: 需要授予辅助功能权限

在设置中开启 **UI 树感知** 功能。

#### 📸 自动截图

发送消息时自动截取屏幕：

- **智能触发**: 没有附带媒体时自动截图
- **全屏捕获**: 截取整个屏幕
- **快速分享**: 无需手动截图
- **可选功能**: 在设置中开启或关闭

在设置中开启 **自动截图** 功能。

#### ⚡ 静默执行命令

AI 生成的命令自动执行，无需手动确认：

- **自动执行**: 命令生成后立即执行
- **提升效率**: 适合频繁执行简单命令
- **安全提示**: 危险命令仍会提示确认
- **可选功能**: 在设置中开启或关闭

在设置中开启 **静默执行命令** 功能。
