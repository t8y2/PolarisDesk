# 快速开始

## 下载

访问 [GitHub Releases](https://github.com/t8y2/PolarisDesk/releases) 下载。

- Windows: `.exe` 安装包
- macOS: `.dmg` 文件
- Linux: `.AppImage` 文件

## 配置

首次启动后：

1. 点击右上角设置
2. 选择 AI 服务商
3. 输入 API Key
4. 选择模型

## 使用

输入问题，按 `Enter` 发送。

## 开发

```bash
git clone https://github.com/t8y2/PolarisDesk.git
cd PolarisDesk
pnpm install
pnpm dev
```
